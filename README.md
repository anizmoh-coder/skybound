# Skybound - Flappy Bird Game

A fun and engaging Flappy Bird game implementation in Python using Pygame.

## Features
- 🐦 Classic Flappy Bird gameplay
- 📊 Score tracking
- 💥 Collision detection
- 🎮 Smooth bird physics
- 🌀 Randomly generated obstacles
- 🎨 Colorful graphics
- ⌨️ Easy controls

## Requirements
- Python 3.7+
- Pygame 2.5.2+

## Installation

1. Clone the repository:
```bash
git clone https://github.com/anizmoh-coder/skybound.git
cd skybound
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python flappy_bird.py
```

## Controls
- **SPACE** - Make the bird jump / Restart after game over
- **ESC** - Quit the game

## Game Rules
- Avoid hitting the green pipes
- Gain 1 point for each pipe you successfully pass through
- Game ends if you collide with a pipe or go out of bounds
- Try to get the highest score!

## Game Mechanics
- **Gravity**: The bird continuously falls due to gravity
- **Jump**: Press SPACE to make the bird jump
- **Pipe Gaps**: Navigate through the gaps between pipes
- **Score**: Increases when you pass a pipe completely
